# Implementation of Ubuntu Virtual Machine on Microsoft Azure with GUI Access using RDP

## Project Description
This project demonstrates the deployment of an Ubuntu Virtual Machine on
Microsoft Azure with secure remote access. GUI-based access is enabled using
Remote Desktop Protocol (RDP), providing both terminal and desktop environment
access to the virtual machine.

## Objectives
- Deploy an Ubuntu Virtual Machine on Microsoft Azure
- Enable secure SSH access to the VM
- Configure GUI-based access using Remote Desktop Protocol (RDP)
- Understand Azure VM provisioning and remote connectivity

## Technologies Used
- Microsoft Azure
- Ubuntu Linux
- Azure Virtual Machines
- SSH
- Remote Desktop Protocol (RDP)

## Project Output
- Ubuntu VM successfully created on Azure
- SSH-based terminal access established
- GUI desktop access enabled via RDP
